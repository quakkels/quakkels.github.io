50 Basic Strongman: Week 6 Day 2

Lower Day

I haven’t worked out for the last few days. So, this week will only have 2 workouts. Today was my lower day, rather than events. I’m not too worried about it because I haven’t been rotating Farmers and Sandbag the last few weeks.

Paused Front Squats

Work up to a 1RM over seven sets. Pause in hole for one second.

Warmup

  • 45x10
  • 95x5

Work

  1. 135x1
  2. 185x1
  3. 225x1
  4. 245x1
  5. 265x1
  6. 295x1
  7. 315x1 PR

18” Deadlift

Work up to a 2RM over five sets.

Warmup

  • 135x5

Work

  1. 225x2
  2. 365x2
  3. 405x2
  4. 455x2
  5. 475x2 PR

I’m pretty happy with these PRs. I feel like I could have gone heavier with both, but it’s probably good that I didn’t so I have room to continue setting PRs as the program continues.

Now, on to the second half of Basic Strongman.
